## 如果你也想拥有和我一样的博客，如何操作？

1. 请拉取 `hexo` 分支代码。
2. 在项目根目录下，执行 `npm install` 安装插件包。
3. 将 `_config.yml` 和 `themes/hexo-theme-matery/_config.yml` 这两个配置文件中的信息改成你自己的信息即可。
4. 执行 `hexo cl` 清除生成文件，然后执行 `hexo s` 开启本地服务。
5. 在浏览器中访问 `localhost:4000` 即可访问项目。
6. 完！
